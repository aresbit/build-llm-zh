# 前言 (Preface)

我一直对语言模型着迷。十多年前，我的 AI 之旅始于一门统计模式分类课程，它催生了我的第一个独立项目：根据歌词检测歌曲情绪的模型与 Web 应用。

快进到 2022 年，随着 ChatGPT 的发布，大语言模型（Large Language Models, LLMs）席卷全球，彻底改变了我们许多人的工作方式。这些模型用途极为广泛——检查语法、撰写邮件、总结长篇文档，凡此种种，不一而足。这源于它们解析和生成类人文本的能力，这一能力在从客服到内容创作、乃至编程与数据分析等技术领域中都至关重要。

顾名思义，LLM 的标志性特征就是"大"——非常大——包含数百万到数十亿的参数。（作为对比，使用传统的机器学习或统计方法，仅需两个参数的小型模型就能以 90% 以上的准确率对鸢尾花数据集进行分类。）然而，尽管 LLM 相比传统方法规模庞大，它们并不一定是黑箱。

在本书中，你将一步一步学会如何构建一个 LLM。读完本书后，你将对 ChatGPT 等平台所用 LLM 的工作原理有扎实的理解。我相信，对每个基本概念及其底层代码建立信心，是取得成功的关键。这不仅有助于修复 bug 和提升性能，还能让你自由尝试新的想法。

几年前，当我开始研究 LLM 时，我不得不通过最艰难的方式学习——翻阅大量研究论文和不完整的代码仓库，才形成了整体的理解。通过这本书，我希望让 LLM 更加亲民，提供一份详尽的逐步实现教程，涵盖 LLM 的所有主要组件和开发阶段。

我坚信，理解 LLM 的最佳方式就是从零编写一个——你会发现这也充满乐趣！

祝阅读和编码愉快！

## 关于本书

**Build a Large Language Model (From Scratch)** 旨在帮助你从零开始理解并创建自己的 GPT 式大语言模型。本书首先聚焦文本数据处理和注意力机制编码的基础，然后引导你一步一步实现完整的 GPT 模型。随后，本书涵盖预训练机制，以及针对特定任务——如文本分类和指令遵循——的微调。读完本书，你将深入理解 LLM 的工作原理，并掌握构建自己模型的技能。虽然你创建的模型在规模上小于大型基础模型，但它们使用相同的概念，是掌握构建最先进 LLM 核心机制与技术的强大教育工具。

### 谁适合读这本书

本书面向希望深入理解 LLM 工作原理并学习从零构建自己模型的机器学习爱好者、工程师、研究人员、学生和从业者。无论初学者还是有经验的开发者，都能运用已有的技能和知识来把握创建 LLM 的概念与技术。

本书的独特之处在于**全面覆盖了构建 LLM 的完整流程**——从处理数据集，到实现模型架构，再到在无标注数据上预训练，最后针对特定任务进行微调。截至目前，没有任何其他资源能够提供如此完整且动手实操的从零构建 LLM 的方法。

要理解本书的代码示例，你应具备扎实的 Python 编程基础。虽然对机器学习、深度学习和人工智能有所了解会更有帮助，但并非必需——LLM 是 AI 的一个独特子领域，即使你相对新手，也能跟上进度。如果你有深度神经网络的经验，某些概念会更为熟悉，因为 LLM 正是建立在这些架构之上。PyTorch 并非先决条件——附录 A 提供了简明入门，帮助你掌握理解全书代码示例所需的技能。

高中数学水平——特别是向量和矩阵运算——有助于探索 LLM 的内部机制，但并不需要高等数学知识来把握书中呈现的核心概念与思想。

最重要的前提是扎实的 Python 基础。有了它，你就已经准备好了去探索 LLM 的迷人世界。

### 本书结构

本书按顺序阅读效果最佳，因为每一章都建立在前面章节引入的概念和技术之上。全书共七章，涵盖 LLM 及其实现的各个核心方面：

- **第 1 章**：高层介绍 LLM 背后的基本概念。探索构成 ChatGPT 等 LLM 基础的 Transformer 架构。
- **第 2 章**：制定从零构建 LLM 的计划。涵盖为 LLM 训练准备文本——包括将文本切分为词和子词 token、使用字节对编码进行高级 token 化、滑动窗口采样训练样本、以及将 token 转换为输入 LLM 的向量。
- **第 3 章**：聚焦 LLM 中使用的注意力机制。从基本自注意力框架开始，逐步过渡到增强自注意力机制，进而实现因果注意力模块，使 LLM 能够逐 token 生成文本，使用 dropout 随机遮蔽注意力权重以减少过拟合，并将多个因果注意力模块堆叠为多头注意力模块。
- **第 4 章**：编码一个能生成类人文本的 GPT 式 LLM。涵盖层激活归一化以稳定神经网络训练、在深度神经网络中添加捷径连接以更有效地训练模型、实现 Transformer 块以创建不同规模的 GPT 模型、以及计算 GPT 模型的参数数量和存储需求。
- **第 5 章**：实现 LLM 的预训练流程。涵盖计算训练集和验证集损失以评估 LLM 生成文本的质量、实现训练函数并对 LLM 进行预训练、保存和加载模型权重以继续训练、以及加载 OpenAI 的预训练权重。
- **第 6 章**：介绍不同的 LLM 微调方法。涵盖准备文本分类数据集、修改预训练 LLM 以进行微调、微调 LLM 识别垃圾消息、以及评估微调后 LLM 分类器的准确率。
- **第 7 章**：探索 LLM 的指令微调流程。涵盖准备监督指令微调数据集、将指令数据组织为训练批次、加载预训练 LLM 并微调以遵循人类指令、提取 LLM 生成的指令回答以供评估、以及评估指令微调后的 LLM。

### 关于代码

为尽可能降低学习门槛，本书所有代码示例均可在 Manning 官网（https://www.manning.com/books/build-a-large-language-model-from-scratch）获取，同时以 Jupyter Notebook 格式托管于 GitHub（https://github.com/rasbt/LLMs-from-scratch）。不用担心卡壳——所有编程练习的解答见附录 C。

本书包含大量源代码示例，既以编号清单形式呈现，也嵌入正文之中。两种情况下，源代码均以 `等宽字体` 显示以区别于普通文本。为适配页面空间，多数原始代码经过了换行和缩进调整。在极少数情况下，清单中会出现续行标记（➥）。此外，当正文对代码进行了描述时，清单中的注释常常被省略。许多代码清单附有注释，突出重要概念。

本书的核心目标之一是可及性，因此代码示例经过精心设计，在普通笔记本电脑上即可高效运行，无需特殊硬件。但如果你有 GPU，某些章节提供了将数据集和模型扩展到利用额外算力的有用提示。

全书使用 **PyTorch** 作为主要的张量和深度学习库，从底层实现 LLM。如果你是 PyTorch 新手，建议从附录 A 开始，其中提供了深入的入门介绍和安装建议。
